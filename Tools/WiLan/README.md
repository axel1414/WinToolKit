## Instalacion 

> Ir a 'Code' y dar click en Download ZIP

```            
            WI-LAN
            
# Si es primera vez: Dar doble click sobre main.bat o directamente en la terminal ejecutarlo.
$ main.bat

# Si ya no es tu primera vez: Ejecuta WiLan.bat
$ wilan.bat
```
